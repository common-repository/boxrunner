BoxRunner shipping method plugin for WooCommerce. Integrates DHL,FedEx,Schenker,UPS and more through BoxRunner.

== Description ==

BoxRunner is a plugin to integrate your woocommerce web shop with BoxRunner warehouse management system (http://boxrunner.net).
Through this plugin you can sync your products to the BoxRunner system and manage your warehouse. It manages products, categorys, picking, packing and shipping your products .Also users can make sales order in the webshop, we process the order in BoxRunner. Furthermore if user adds a new product in warehouse side, it automatically sync to the web shop after the configuration is successfully done. This plugin is really helpful for the user to manage their web shop and warehouse.


= Main Functionalites =

* Manage warehouse.
* Sync product and orders when creating from the web shop to the BoxRunner.
* Sync all product at once.
* Print shipping document to the each order.

= Requires =
* BoxRunner for WooCommerce requires the WooCommerce plugin as well as an account on BoxRunner. You will need to enter your BoxRunner Consumer Key and Consumer Secret to activate it. The account is free of charge and is used to administrate your process.

* Register your free account on [BoxRunner](http://subscription.boxrunner.net/index.php "http://subscription.boxrunner.net/index.php")

* PLEASE NOTE that BoxRunner currently only supports Swedish webshops!

== Installation ==

= Minimum Requirements =

* WordPress 3.8 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater
* WooCommerce 2.4.9 or below 
* WooCommerce PDF Invoices & Packing Slips 1.5.23 or greater

= Automatic installation =

* Select 'Add new' in the 'Plugins' menu
* Find and install the latest Boxrunner plugin for WooCommerce.
* Activate the plugin.
* Setup the module under  'Settings' / 'BoxRunner'

= Manual Installation =

* Upload the folder boxrunner manually to the /wp-content/plugins/ directory through FTP.
* Activate the plugin through the 'Plugins' menu in WordPress
* Setup the module under 'Setting' / 'BoxRunner' 

== Frequently Asked Questions ==

= How to connect the BoxRunner? =

Go to wordpress settings and check whether given fields are filled correctly and save it. If it is not connected, please contact Boxrunner support (http://boxrunner.net/#contact).
